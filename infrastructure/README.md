# ATLAS Platform - Infrastructure

This directory contains infrastructure-as-code and deployment configurations.

## Structure

```
infrastructure/
├── terraform/           # Terraform configurations (AWS/GCP)
├── kubernetes/          # Kubernetes manifests
├── docker/             # Docker configurations
├── prometheus/          # Prometheus configuration
└── grafana/            # Grafana dashboards
```

## Docker

See `docker-compose.yml` in the project root for local development.
